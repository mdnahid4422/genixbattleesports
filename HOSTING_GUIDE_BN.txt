আপনার ওয়েবসাইটটি ১ টাকাও খরচ না করে ফ্রিতে লাইভ করার জন্য নিচের ধাপগুলো অনুসরণ করুন। আমরা এখানে Vercel ব্যবহার করার পরামর্শ দিচ্ছি কারণ এটি সবচেয়ে ফাস্ট এবং সহজ।

ধাপ ১: GitHub এ কোড আপলোড করা
-------------------------------------------------------
Vercel বা অন্য যেকোনো প্ল্যাটফর্মে ফ্রিতে হোস্ট করতে হলে আপনার কোডগুলো GitHub এ থাকা জরুরি।
১. github.com এ একটি অ্যাকাউন্ট খুলুন।
২. একটি নতুন "Repository" তৈরি করুন (নাম দিন: genix-esports)।
৩. আপনার কম্পিউটারের কোডগুলো সেখানে Push করুন (অথবা 'Upload files' অপশন ব্যবহার করে সব ফাইল আপলোড করে দিন)।

ধাপ ২: Vercel এর সাথে কানেক্ট করা
-------------------------------------------------------
১. vercel.com এ যান এবং আপনার GitHub অ্যাকাউন্ট দিয়ে লগইন করুন।
২. "Add New" বাটন থেকে "Project" এ ক্লিক করুন।
৩. আপনার GitHub এর সেই Repository টি (genix-esports) সিলেক্ট করুন।
৪. "Framework Preset" এ 'Vite' অটোমেটিক আসার কথা, না আসলে Vite সিলেক্ট করুন।
৫. "Deploy" বাটনে ক্লিক করুন। ২-৩ মিনিটের মধ্যে আপনার ওয়েবসাইট লাইভ হয়ে যাবে!

ধাপ ৩: নিজের ডোমেইন সেটআপ (ঐচ্ছিক)
-------------------------------------------------------
Vercel আপনাকে একটি ফ্রি লিঙ্ক দিবে (যেমন: genix-esports.vercel.app)। যদি আপনার নিজের .com বা অন্য কোনো ডোমেইন থাকে, তবে Vercel এর ডোমেইন সেকশনে গিয়ে সেটি ফ্রিতে কানেক্ট করতে পারবেন।

গুরুত্বপূর্ণ নোট (LocalStorage সংক্রান্ত):
-------------------------------------------------------
যেহেতু আপনি ওয়েবসাইটটি "Static" রাখতে বলেছেন এবং Gemini/Firebase বাদ দিয়েছেন, তাই বর্তমানে সব ডাটা (যেমন: অ্যাডমিন প্যানেলে করা চেঞ্জ) শুধুমাত্র আপনার নিজের ব্রাউজারের "LocalStorage" এ সেভ হবে। অন্য কোনো ইউজার তাদের মোবাইল থেকে ওয়েবসাইট দেখলে আপনার করা চেঞ্জ দেখতে পাবে না যদি না আপনি ডাটাবেস (যেমন: Firebase) আবার যুক্ত করেন।

যদি আপনি চান যে আপনার অ্যাডমিন প্যানেলের পরিবর্তন সবাই দেখুক, তবে আপনাকে "Static" এর বদলে আবার ডাটাবেস বা গুগল শিট সিঙ্ক সিস্টেম চালু করতে হবে। বর্তমানে এটি একটি পার্সোনাল ড্যাশবোর্ড হিসেবে কাজ করবে।
